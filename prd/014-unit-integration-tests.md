- Updates or adds unit/integration tests for all modified components.

## Requirements

- Place unit/integration tests next to modified files.

- High coverage and all tests pass.
- Tests are located next to implementation files.

- [ ] Ensure all documentation (`SPEC.md`, `README.md`) are updated and remain consistent with the implemented code changes.
